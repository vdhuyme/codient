# 🎉 VDHuyMe API - License Management Magic! ✨

Xin chào! Welcome to the most awesome license management API you'll ever meet! 🚀

## 🌟 What's This All About?

This is not just any API - it's your friendly neighborhood license manager that helps you:

- 🔑 Create shiny new licenses with custom durations
- ✅ Verify license tokens with style
- ⏰ Handle time-based activations like a boss
- 🛡️ Keep everything secure and sound

## 🎯 Features

- 🎨 Beautiful REST API design
- 🔒 JWT-based license tokens
- ⚡ Lightning-fast responses
- 📅 Time-based license validation
- 🛑 Smart error handling
- 📝 Request validation

## 🚀 Quick Start

1. Clone this bad boy:

```bash
git clone [your-repo-url]
```

2. Install the goodies:

```bash
npm install
```

3. Fire it up:

```bash
npm run dev
```

## 🎮 API Endpoints

### Create License 🎫

```http
POST /v1/license
```

### Verify License 🔍

```http
GET /v1/license?token=your_token
```
